# MAX7219 Raspberry Pi Clock

32x8 LED matrix clock for Raspberry Pi using luma.led_matrix.

## Features

- HH:MM clock
- CPU temperature
- Date ticker
- Auto brightness (day/night)
- Seconds progress bar
- Animations

## Hardware

- Raspberry Pi
- MAX7219 32x8 LED matrix
- SPI connection

## Installation

```bash
pip install luma.led_matrix
